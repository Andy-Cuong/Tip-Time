Tip Time - Solution Code
=================================

Solution code by learner for the [Android Basics with Compose](https://developer.android.com/courses/android-basics-compose/course): Tip Time app.


Introduction
------------
The Tip Time app contains various UI elements for calculating a tip,
teaching about user input, and State in Compose. It also includes local
(unit) test and instrumentation (UI) test.


Pre-requisites
--------------
* Experience with Kotlin syntax.
* How to create and run a project in Android Studio.


Getting Started
---------------
1. Install Android Studio, if you don't already have it.
2. Download the sample.
3. Import the sample into Android Studio.
4. Build and run the sample.
